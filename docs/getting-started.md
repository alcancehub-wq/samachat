# Getting Started

## Install
- Ensure Node.js and pnpm are installed.
- From the repo root, run `pnpm install`.

## Development
- Start the stack with `pnpm dev`.
- Web runs on port 3000 and API on port 3001 by default.

## Validation
- Run the P3 validation suite with `pnpm run validate:p3`.

## Tests
- Run workspace tests with `pnpm -r test`.
- For a single package, use `pnpm --filter <package> test`.
